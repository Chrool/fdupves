fdupves - find duplicate video/image files

* Description

Searches the given path for duplicate video or image files. Such files are found by Perceptual Hash of every file.

* Requirement

 - Gtk2: http://www.gtk.org/
 - FFmpeg: http://ffmpeg.org/
